# IjkVideoPlayer
基于Bilibili/ijkplayer的Android播放器。
更清晰的代码逻辑，更灵活易用的API，更易拓展的UI结构。

- 支持直播和视频
- 支持HTTPS
- 支持非WIFI网络下的状态提醒
- 支持多个视频流切换
- 支持播放完毕的重新播放
- 支持音量和亮度的拖等手势，支持进度的拖动手势
- 支持友好的出错提示及重新获取按钮
- 支持加速传感器的方向监听
- 支持屏幕方向锁定
